* Description
[[https://github.com/conao3/smear-cursor.el][https://raw.githubusercontent.com/conao3/files/master/blob/smear-cursor.el/smear-cursor.gif]]

Renders smear trail behind the cursor.

* Install
Sample install code using [[https://github.com/conao3/leaf.el][leaf.el]].

#+begin_src emacs-lisp
  (leaf smear-cursor :ensure t)
#+end_src

*** Running test
Below operation flow is recommended.
#+begin_src shell
  git branch [feature-branch]       # Create branch named [feature-branch]
  git checkout [feature-branch]     # Checkout branch named [feature-branch]

  # <edit loop>
  emacs smear-cursor.el             # Edit something you want

  make test                         # Test smear-cursor
  git commit -am "brabra"           # Commit (auto-run test before commit)
  # </edit loop>

  gh repo fork                      # Create fork at GitHub
  git push [user] [feature-branch]  # Push feature-branch to your fork
  gh pr create                      # Create pull-request
#+end_src
